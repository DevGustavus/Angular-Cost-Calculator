A calculadora permite que você registre seus gastos junto com o nome da transação, sua descrição e valor.

A calculadora é uma ferramenta essencial para gerenciar suas finanças pessoais. Com ela, você pode ter um controle preciso dos seus gastos, categorizá-los e entender melhor para onde seu dinheiro está indo.

Um projeto Angular :)

Este projeto foi realizado com intuito educativo.

Em um momento de criatividade, decidi desenvolver um projeto simples sobre uma calculadora financeira. Este projeto me ajudou a treinar e aprimorar minhas habilidades em Angular , especialmente em componentização de elementos e consumo de API.

Apresento-lhes um sistema simples cujo banco de dados é um arquivo JSON. Este sistema é executado através do "JSON server", fornecendo uma rota para que o sistema Front-End o consuma.
